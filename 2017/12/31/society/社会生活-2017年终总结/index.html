我的 2017 总结

发表于 2017-12-31 | 分类于 社会生活
“时间过得太快，妈逼的，活着活着就老了。”
冯唐

写在前面

这是我第一次认真地以文章形式做一年的总结。

上学时我从不屑用课堂笔记本，觉得抄笔记的人都太愚笨。但在生活中，我一直都在记录东西，几个字，几个日期，几个小目标，总是要写一下，如果不这样做，我相信人很快会变成猩猩。

很遗憾，这些记录像卫生纸一样，临时用一下就扔到一边，最后都被时间淘洗得稀烂或不见踪影。

马上 2018 ，2018 - 1988 = 30。三十而立，孔丘说的。孔丘距今 2000 年有余，文化、科技、观念早已今非昔比，所以我觉得对于现在的我，并不适用。奈何认知框架的作用（“别想那只大象”），我们无法摆脱根深蒂固的观念的影响，所以 30 岁的到来对我来说还是有点别扭的。

既然古人 30 岁都能人格自立、学识自立、事业自立，那咱也不能不如古人吧？

做个正式的总结，已示对自己拥有过的时光的尊重。

2016 年底列的计划

由于还未养成良好的精细的量化习惯，所以当时只罗列了大致方向，并不具体，大概包括如下几块（重要性和顺序无关）：

工作技能
1- 深耕计算机语言，有自己的产品（app，网站等）。
2- 整理知识，以博客方式记录笔记。

业余爱好
1- 拓展知识面：《得到》，《一块听听》，《TED演讲》，新生大学分享，别人博客，知乎等。
2- 读书：计算机类、方法论类、社科类、经典人文类，冷门小众的尽量少看。
3- 影视：IMDB TOP 250经典。
4- 绘画：练习手绘板、画几张大画送朋友(自己爱装逼，嘴贱，口头已欠了好几张)。
5- 摄影：基本知识巩固 + 操作练习；美化整理已有照片。
6- 吉他：弹奏一首完整的曲子。
7- 英语：读原版书，背常见生词。

生活事故
1- 研究装修的书，自己出装修方案。
2- 结婚。
3- 父母老了，记得多关注他们的变化。
4- 旅游。

身体素质
1- 骑山地车轧北京马路，用 Keep App。
2- 保证睡眠时间。

2017 年感想总结

依然逐块总结：

工作技能
终于从土建行业彻底抽身出来，投入了高新技术驱动的计算机互联网行业。做这个决定耗费了我很多心力，现在终于有了个结果。耗一年时间，总算对计算机领域有了个大的全的框架理解，知道自己身处的坐标。对身份不再焦虑，但对前景依然焦虑。

社会分工越来越细，对什么都感兴趣的我，在 30 岁之际需要想一想自己的看家本领了，否则终将因为竞争力太弱而导致竞争压力一直存在。

既然投身了计算机互联网，那就精通下去吧，不能再因为“差不多”就不再追究技术细节，不能长板不够长。虽然计算机语言和技术日新月异，但基本思想和基础架构十几年甚至几十年都不会变，所以，重点是基础知识，了解最新技术动态 即可，不要跟风（人工智能、大数据…）。

业余爱好
这一年，到处都是“终身学习”、“知识付费”，以罗振宇为首的贩卖知识的一群人给一般普通人造成了不小的压力。不过我觉得，焦虑不是坏事：焦虑、浮躁，证明你不是一潭死水，你一直会有涟漪，甚至会产生波澜，生命因此而可能会更加精彩。再不折腾，就老了。

对于到处涌现的知识服务商，首先，应该有所动有所不动：自己大脑的认知框架不能随意动，认知的碎片随便动。

什么意思呢？就是给自己大脑分区：哲学，社科，艺术，科技，商业… 这些区就这几类；每天接触到的学习到的新鲜东西随便是什么，都贴上对应的标签扔进对应的大脑分区，让大脑自己化学反应去吧。

其次，要有 独立思考能力：碎片要不要捡起来？向高中备考似的天天上课，天天看幻灯片，天天听老师叨逼叨，甚至到大学了都没有逃过课的人，肯定是个呆子，是个废物。

最后，最重要的，要有目标，要有执行能力。否则，看了那么多书，依然过不好一生…

关于读书和看电影，我真不禁感叹：

当今人类每天制造（很少数称得上创造）的东西太他妈的多了，我们太容易被现在这个才几十年历史的信息时代的洪流裹挟，进而不能自已，丢掉自我。

上面我罗列的图书和影视，经典和口水平分秋色。经典是根，是躯干，牵一发动全身；口水是枝杈，是树叶，一岁一枯荣。

伟大的作品，都经过伟大的读者品味过。我不要做无脑跟风读者，所以，我还有很大的调整空间。

关于画画，这一年学会了用数位板，很不错。对于这种抽象万变的活动，我想，只要保持画的数量即可。对我来说，画画的重要意义在于放松，调整心情，偶尔装个小逼。

摄影，是一个有门槛的业余爱好，很容易浅尝辄止，幸亏我不是那样的人。不过一年内，我似乎毫无进步。新的一年，还是需要练习基本功。

吉他，三天不摸，技术就大打折扣，可远观不可亵玩，真是让我爱恨交错。不知道天天摸一小会儿，坚持一年，效果大不大。拥有一整块时间，真的是太奢侈了。

英语，在生活工作中真是悲剧的毫无场景。好吧，要刻意看美剧，刻意读英文原版书目。

生活事故
上面罗列的，似乎都不是我很刻意就完成了的。想了想原因，也许是没有明确的小目标吧，所以无法执行。

身体素质
毫无用心去做，野蛮生长，自由放任。最知道如何操作的东西，却最懒得去做，最需要时间积累的习惯，是那么难养成。

2017 年惊喜

最后，要提一下惊喜。我们要有自我主观放大惊喜的能力，哈，这样自己才会知道自己原来这么幸福。

惊喜1：偶然尝试下，部分美剧可以不要字幕…希望我继续坚持，让此项技能和骑自行车一样，掌握了，就永远掌握了。

惊喜2：学会了魔方。嗨，没有什么不可以。

惊喜3：家里多了一个成员。养宠物，栽植物，温暖我们在工业社会下冷冰冰的机器。

2018 年初列的计划

2017 年整体差强人意，未来仍然希望满满。嗯，有希望，是一种很健康的状态。

工作技能
1- 将 Rails 教程梳理一遍，形成日志系列。
2- 过一遍 C ，C++ ，C# ，设计模式。
3- 研究 Android 和小程序，发布一个自己的作品（去年未完成好的或今年可能新产生的）。
4- 每月至少两篇博客。当无物言之时，证明需要“输入倒逼输出”。

业余爱好
1- 拓展知识面：每天听本书，总结-分类-记录；TED 演讲每月 3 个，一年 36 个。
2- 读书：保持 2017 状态，另外多些认真，多些总结和感悟并记录；重要的 《CSAPP》。
3- 影视：保持 2017 状态，多看经典，摒弃口水。
4- 绘画：纸质 ≧ 1，手绘 T 恤衫 = 1，数位板 3 。
5- 摄影：通读操作手册，打好框架。
6- 吉他：每天摸一会儿，完成 2017 遗留的《成都》。
7- 英语：看一本原版，追更《权力的游戏》等基础上再看一部经典美剧，过一遍有道生词。
最后，就像冯唐《大器》说的那样，买几个好玩意儿，顺手的吉他、看大片的投影仪、大屏高配的手机…

生活事故
1- 继续装修房子，追求逼格。
2- 给老爸老妈买东西，多打电话多聊天，想办法让老爸捡起书法，老妈捡起小手艺活儿，不能天天喝酒看电视。
3- 自驾游 ≧ 1 次。
4- 生小孩儿…
5- 投资？数字币？

身体素质
1- 山地车越野 ≧ 1，Keep 每天任务，羽毛球🏸️。（有氧运动很重要，因为关乎性福。）
2- 暂无好的硬性指标，尽量靠体能消耗或睡前听书倒逼休息。
